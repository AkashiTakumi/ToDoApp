# ToDoApp
KobeSpiralのSpring練習アプリ

## 使い方
- 以下の二つからどっちかで
    - springをToDoApplication.javaから実行．
        - http://localhost:18080/
    - build/libs/ToDoApp.warをTomcatないのWebapps内にデプロイして，tomcatを起動する．
        - http://localhost:18080/ToDoApp

- adminの管理者画面からユーザ登録
    - これは明石が行うもの

## 出来ないこと
- ちゃんとしたセキュリティ
- doneのreopen
